# React Examples

Each example focuses on a single React concept.

| Example       | Concept    |
| ------------- | ---------- |
| 01-jsx        | JSX        |
| 02-components | Components |
| 03-props      | Props      |
| 04-state      | State      |
| ...           | ...        |

To run an example:

1. Open `src/App.jsx`
2. Import the desired example
3. Run

```bash
npm run dev
```
